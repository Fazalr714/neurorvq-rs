## 🚀 What this app does

neurorvq-rs turns biosignal data into tokens you can use for analysis and model input. It focuses on EEG, ECG, and EMG data, and it runs in Rust with Burn ML for local inference.

Use it when you want to:
- load biosignal files on Windows
- tokenize EEG, ECG, or EMG data
- run inference on your own computer
- work with a Rust-based tool for signal processing
